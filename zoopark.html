Zoopark Python Project


Written by Buffal0x

This is the Zoopark Python project


---------------------------------

It is written in Python and acts as a "Zoo Simulator" or Game
It took some time to make this project as there were a few new methods of doing things.
Overall the project took me about 48h straight to finish.
The game has features such as feeding and playing with the animals.




After you've added an animal it will prompt you that the animal has been created or error will occur.
As you might see there are a few menu options to choose from.
If you choose to feed or remove an animal, you don't have to select the animal by name
but rather by ID number which makes it all super easy to manage.




This is basically the whole game.
